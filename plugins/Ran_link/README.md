# Ran_link 插件文档

## 插件概述
**名称**: 友情链接 (Ran_link)  
**作者**: RanUI  
**版本**: 1.0.0  
**描述**: 友情链接管理系统。支持前台自助申请，后台审核通过后自动显示。

## 功能特性
1.  **自助申请**: 提供申请页面供其他站长提交友链。
2.  **审核机制**: 管理员后台审核 (Pending -> Approved)。
3.  **随机展示**: 首页底部随机展示已通过的友链。

## 数据库表
*   `links`: 存储链接信息及状态。

## 外部依赖与关联
*   独立插件。

## 挂载点 (Hooks)
*   `admin_sidebar_extension`: 注册 "友情链接" 菜单。
*   `theme_footer_links`: 在页脚模板位置渲染友链列表。
